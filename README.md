```console

```
 
### Run the game in DOSbox

```console
 link pong
```

## Controls
1. Main menu
- `S`, `s`- Single Player,
- `M`, `m`- Multi Player ,
- `E`, `e`- Exit,

2. During the game
- `O`, `o`-  MOVE_RIGHT_PADDLE_UP,
- `L`, `l` -  MOVE_RIGHT_PADDLE_DOWN,
- `W`, `W` -  MOVE_LEFT_PADDLE_UP,
- `S`, `s`-  MOVE_LEFT_PADDLE_DOWN,

3. Game over
- `R`, `r` - Play again,
- `E`, `e`-  Go to main menu,
